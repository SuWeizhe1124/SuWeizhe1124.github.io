SEO 是指透過優化網站內容與體質的方式，讓排名可以自然排序到第一頁甚至是前三名，而這些方式包括了撰寫優質的網頁Title/Description、豐富文案內容、圖片優化、結構化資料運用、網站讀取速度提升、使用者體驗設計等等，

wordpress-前端網站設計

頁是由前端、後端兩大部分組成，前端採集輸入資訊，後端進行資訊處理。
簡言之，我們平常看到的網頁就是前端，而後端程式則構成了後台管理系統。
網頁前端設計，由三大元素構成：HTML、CSS、JavaScript。

  • HTML ：網頁內容的描述語言，負責建立網頁的主結構。
  • CSS ：網頁外觀形態的描述語言，負責美化網頁。
  • JavaScript ：一門函式先行的直譯式程式語言，經常用在呈現網頁動態效果。負責描述網頁如何與用戶互動。
    透過HTML以及CSS的相互合作，就能將網頁結構化，做成我們希望的視覺呈現方式，再藉由JavaScript，
    讓我們可以在只能閱覽文字與影像的網頁中，加入動態效果並與使用者互動。
wordpress - 電商購物網站＋API串接＋系統串接

電商「金流」串接簡單來說，即是指商家向消費者收取費用的方式，常見的像是現金付款、電子支付、 行動支付、信用卡付款等， 而相較於實體的交易，網路電商的金流方式更加繁雜，必須透過「金流商」居中協調，才能將網路上一筆一筆的訂單費用彙整，進而撥款 匯入店家的帳戶。

一、消費者有權要求退(換)貨的情形
(一)不具備合理應有的功能(如檔案出現問題無法打開等問題)。

(四)商品並未具備企業經營者所稱的功能。

(一)購買前企業經營者已告知或明示商品的缺陷。
(二)購買前已檢查商品，卻因自己的重大過失而未發現顯而易見的缺陷，且企業經營者沒有保證商品無缺陷時。
